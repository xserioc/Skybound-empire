# ✈️ Skybound: Global Aviation Empire

**Skybound** is a web-based flight tycoon built with Three.js. Start with your grandfather's vintage Cessna 172 in New York City and build a multi-national cargo and passenger empire.

## 🗺️ The Journey
The game follows a global expansion path. You must level up and unlock permits to fly into new sectors:
1. **New York (NYC)** - The Start
2. **Los Angeles (LAX)** - Level 10
3. **Tokyo (TYO)** - Level 25
4. **Beijing (PEK)** - Level 40
5. **Abu Dhabi (AUH)** - Level 60
6. **Paris/Rome (EUR)** - Level 80
7. **London (LHR)** - Level 100

## 🎮 Controls
- **[P]**: Toggle Engine Start/Stop
- **[W/S]**: Throttle Up / Brake
- **[A/D]**: Steer (Taxi) / Bank (Flight)
- **[Shift/Ctrl]**: Pitch Up / Pitch Down

## 🛠️ Technical Setup
This project is designed to run in **GitHub Codespaces**.
- **Engine:** Three.js (via CDN)
- **Assets:** GLTF/GLB Models
- **Styles:** Custom CSS3 for Aviation UI

## 📜 The Story
You’ve inherited more than just a plane; you’ve inherited a legacy. With $45,000 and an old Cessna 172, you must prove to Artie (your lead mechanic) and the world that you can handle the "45-degree comfort rule" and the high-pressure world of international flight.
